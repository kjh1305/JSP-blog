### JSP-blog
</br></br>
부트스트랩 무료 템플릿을 활용한 blog  </br></br>
MVC MODEL 2 디자인 패턴 사용  
**TOLL**  
-Intellij, JSP/SERVLET, ORACLE, TOMCAT  
</br></br>
**기능**  
-회원가입 기능  
-로그인 기능 (session,cookie(id,pw 저장 활용))  
-회원조회 및 로그인시 회원정보 수정 삭제(계정확인) 가능    
-블로그 CRUD(삭제시 비밀번호 확인)  
-페이징 기능
